# 1.6.0 Release Notes

## ✨ 特性
- 新增订阅功能，支持 AutoProxy/SwitchyLegacy/SwitchyOmega/PAC 的订阅格式
- 支持订阅的自动更新，目前自动更新支持粒度为 1分钟/6小时/12小时/1天
- 支持反转订阅的匹配和绕过规则，用于支持各种白名单和黑名单的代理订阅
- 新增代理和场景的唯一 ID 生成机制
- 新增手动模式下切换场景时自动选择并应用代理
- 支持在代理配置中快速查看从订阅规则中提取的匹配和绕过的规则列表
- 优化订阅规则计数显示的国际化支持

## 🐛 问题修复
- 无
