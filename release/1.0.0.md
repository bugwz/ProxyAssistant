# 1.0.0 Release Notes

## ✨ 特性
- 支持配置多种代理协议(HTTP/HTTPS/SOCKS5), 并支持代理协议认证
- 支持代理连接探测, 包括单个代理连接测试和全部代理连接批量测试
- 支持使用多种代理模式(手动代理模式和自动代理模式)
- 支持使用主题配置(浅色主题，深色主题以及可指定时间的自动主题切换)
- 支持两种数据存储方式(Google账户数据联网存储方式和本地数据存储方式)
- 支持多语言(简体中文,繁体中文,英语,日语,法语)
- 支持自动化发布流程文档 (AI Prompt)
- 优雅边界的操作界面
