# 1.1.0 Release Notes

## ✨ 特性
- 新增 5 种语言支持（支持简体中文、繁体中文、日语、法语、德语等）
- 采用系统语言设置自动初始化界面语言
- 添加 Chrome 和 Firefox 浏览器适配清单文件
- 新增 Firefox XPI 安装包构建脚本
- 重构构建流程，分离浏览器 Worker 实现
- 新增代理状态检测功能，优化弹窗交互体验
- 优化不同模式下的颜色显示方案
- 优化数据同步逻辑，支持状态徽章显示
- 优化表单输入切换交互体验
- 禁用模式下隐藏状态徽章显示

## 🐛 问题修复
- 修复后端界面主题设置不生效的问题
- 修复 Firefox 包验证错误问题
- 修复 updateStatusDisplay 函数作用域问题
- 修复自动模式下代理异常问题
- 修复手动模式下代理节点选择失败问题
- 修复 Chrome 存储同步错误处理
- 修复手动模式下状态徽章不显示问题
- 修复 Firefox 清单文件警告
- 修复发布工作流错误

## 📋 其他
- 添加隐私政策文档
