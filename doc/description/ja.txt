ChromeとFirefox向けの強力なブラウザプロキシ管理拡張機能。

【コア機能】

🔌 マルチプロトコル対応
• HTTP、HTTPS、SOCKS5、SOCKS4 プロキシプロトコル
• プロキシ認証対応（ユーザー名/パスワード）

🔄 3つのプロキシモード
• 無効モード：プロキシをオフにして直接接続
• 手動モード：一クリックで希望のプロキシに切り替え
• 自動モード：URLルールに基づいてスマートにプロキシをマッチング

📋 柔軟なURLルール
• 異なるウェブサイトに異なるプロキシを設定
• ワイルドカードマッチングに対応
• 直接接続またはプロキシ失敗時のフォールバック戦略を設定

📥 サブスクリプションルール
• URLからプロキシルールをインポート
• 複数のフォーマットに対応：AutoProxy、SwitchyOmega、SwitchyLegacy、PAC
• サブスクリプションURLの認証に対応

⏰ 自動スケジュール更新
• サブスクリプションURLから最新のプロキシリストを自動取得
• 手動更新なしで常に設定を最新に維持
• 更新間隔のカスタマイズ対応（1分〜24時間）
• 複数のサブスクリプションフォーマットに対応：AutoProxy、SwitchyOmega、PAC

🧪 プロキシテスト
• 一クリックでプロキシの可用性をテスト
• 応答レイテンシを表示（緑<500ms / オレンジ≥500ms / 赤失敗）
• すべてのプロキシを一括テスト

🌙 ダーク/ライトテーマ
• 手動切り替えまたは自動時間ベース切り替え

🎬 シナリオモード
• カスタムシナリオモードに対応

☁️ データ同期
• Chrome/Firefoxアカウントのクラウド同期
• ローカルファーストストレージ、データ損失なし
• 設定のインポート/エクスポート対応

🌍 10言語対応
簡体字中国語、繁体字中国語、英語、日本語、フランス語、ドイツ語スペイン語、ポルトガル語、ロシア語、韓国語

【ユースケース】

• 複数のプロキシを素早く切り替え
• スマートルーティング（国内は直接接続/特定サイトはプロキシ）
• プロキシポールのレイテンシテストと選別
• チーム設定の共有
• プロキシサブスクリプションの自動更新（手動メンテナンス不要）

【選ばれる理由】

✓ クリーンなインターフェース、直感的な操作
✓ 豊富な機能でありながら軽量
✓ ローカルストレージ、安全で信頼性が高い
✓ クロスブラウザ対応
✓ 自動更新、メンテナンス不要
✓ 完全無料

【権限について】
この拡張機能は"alarms"権限を使用してスケジュールに従ってプロキシサブスクリプションリストを自動更新し、プロキシ設定が最新であることを確保します。ユーザーデータは収集されません。
