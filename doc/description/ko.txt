Chrome와 Firefox를 위한 강력한 브라우저 프록시 관리 확장 프로그램.

【주요 기능】

🔌 다중 프로토콜 지원
• HTTP, HTTPS, SOCKS5, SOCKS4 프록시 프로토콜
• 프록시 인증 지원 (사용자 이름/비밀번호)

🔄 세 가지 프록시 모드
• 비활성화 모드: 프록시 끄기, 직접 연결
• 수동 모드: 원클릭으로 선호하는 프록시로 전환
• 자동 모드: URL 규칙 기반 스마트 프록시 매칭

📋 유연한 URL 규칙
• 다른 웹사이트에 다른 프록시 구성
• 와일드카드 매칭 지원
• 직접 연결 또는 프록시 실패 시 폴백 전략 설정

📥 구독 규칙
• URL에서 프록시 규칙 가져오기
• 여러 형식 호환: AutoProxy, SwitchyOmega, SwitchyLegacy, PAC
• 구독 URL 인증 지원

⏰ 자동 예약 새로고침
• 구독 URL에서 최신 프록시 목록 자동 가져오기
• 수동 업데이트 없이 항상 최신 구성 유지
• 사용자 정의 새로고침 간격 (1분~24시간)
• 여러 구독 형식 지원: AutoProxy, SwitchyOmega, PAC

🧪 프록시 테스트
• 원클릭으로 프록시 가용성 테스트
• 응답 지연 시간 표시 (녹색 <500ms / 주황색 ≥500ms / 빨간색 실패)
• 모든 프록시 일괄 테스트

🌙 다크/라이트 테마
• 수동 전환 또는 자동 시간 기반 전환

🎬 시나리오 모드
• 사용자 정의 시enario 모드 지원

☁️ 데이터 동기화
• Chrome/Firefox 계정 클라우드 동기화
• 로컬 우선 저장소, 데이터 절대 손실 없음
• 구성 가져오기/내보내기 지원

🌍 10개 언어 지원
중국어 간체, 중국어 번체, 영어, 일본어, 프랑스어, 독일어, 스페인어, 포르투갈어, 러시아어, 한국어

【사용 시나리오】

• 여러 프록시 간 빠른 전환
• 스마트 라우팅 (국내는 직접 연결 / 특정 사이트는 프록시)
• 프록시 풀 지연 시간 테스트 및 필터링
• 팀 구성 공유
• 프록시 구독 자동 업데이트 (수동 유지보수 불필요)

【왜 우리를 선택하세요】

✓ 깔끔한 인터페이스, 직관적인 조작
✓ 풍부한 기능但是 부피过大하지 않음
✓ 로컬 저장소, 안전하고 신뢰할 수 있음
✓ 크로스 브라우저 지원
✓ 자동 업데이트, 유지보수 제로
✓ 완전 무료

【권한】
이 확장 프로그램은 "alarms" 권한을 사용하여 예약에 따라 프록시 구독 목록을 자동으로 새로고침하여 프록시 구성이 항상 최신 상태를 유지하도록 합니다. 사용자 데이터는 수집되지 않습니다.
